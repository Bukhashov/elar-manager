# ELAE MANAGER

| № | Название пакеты | Описание | Команда установки |
|:--- |:---- |:---- |:----:|
| 1. | dotenv | Cчитывает пары ключ-значение из файла .env и может устанавливать их в качестве переменных среды. Это помогает при разработке приложений, основанных на 12-факторных принципах. | pip install python-dotenv |
| 2. | pymongo | Дистрибутив PyMongo содержит инструменты для взаимодействия с базой данных MongoDB на Python. Пакет bson представляет собой реализацию формата BSON для Python. Пакет pymongo представляет собой собственный драйвер Python для MongoDB, предлагающий как синхронные, так и асинхронные API. | pip install pymongo |
| 3. | watchdog | Утилиты для мониторинга событий файловой системы | pip install watchdog |
| 4. | cbsvibpyirbis | Универсальное клиентское программное обеспечение для библиотечной системы автоматизации IRBIS64 | pip install cbsvibpyirbis |
| 5. | PIL | Эта библиотека обеспечивает обширную поддержку форматов файлов, эффективное внутреннее представление и довольно мощные возможности обработки изображений. | pip install pillow |
| 6. | ollama | Предоставляет самый простой способ интеграции вашего проекта на Python 3 с Ollama. | pip install ollama |
| 7. | fitz | высокопроизводительная библиотека Python для анализа, обработки, преобразования и извлечения данных из PDF-документов и других форматов (XPS, EPUB, CBZ) | pip install fitz | 
```
pip install dotenv pymongo watchdog cbsvibpyirbis pillow ollama fitz
```
